\# Exercise 2 - E-Commerce Search Function



\## Aim



To implement an e-commerce search function using Linear Search in C#.



\## Description



In an e-commerce platform, users search for products from a list of available products.  

This exercise demonstrates how Linear Search can be used to find a product from a product list.



\## Algorithm Used



Linear Search



\## Output



Product Found at Position : 4



\## Observation



The search function successfully located the requested product using Linear Search.



\## Time Complexity



Best Case : O(1)



Worst Case : O(n)



\## Space Complexity



O(1)
