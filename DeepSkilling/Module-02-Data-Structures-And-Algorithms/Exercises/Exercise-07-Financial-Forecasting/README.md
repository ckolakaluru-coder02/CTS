\# Exercise 7 - Financial Forecasting



\## Aim



To implement a financial forecasting program using recursion in C#.



\## Description



Financial forecasting is used to estimate future investment value based on present value, annual growth rate, and number of years.



This exercise calculates the future value using recursive compound growth.



\## Formula



Future Value = Present Value × (1 + Growth Rate) ^ Number of Years



\## Input



Present Value = ₹10000  

Growth Rate = 10%  

Years = 5



\## Output



Future Value: ₹16105.10



\## Observation



The recursive forecasting algorithm successfully calculated the future investment value after 5 years with a 10% annual growth rate.



\## Time Complexity



O(n)



where n = number of years.



\## Space Complexity



O(n)



due to recursive function calls.
