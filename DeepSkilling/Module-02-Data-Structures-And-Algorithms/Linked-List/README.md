\# Linked List



This folder contains C# implementations of linked list data structures.



\## Topics Covered



\- Singly Linked List

\- Doubly Linked List



\## Linked List



A linked list is a linear data structure where elements are stored in nodes. Each node contains data and a reference to the next node.



\## Advantages



\- Dynamic size

\- Easy insertion and deletion

\- Efficient memory usage



\## Disadvantages



\- No direct access using index

\- Extra memory required for storing references
