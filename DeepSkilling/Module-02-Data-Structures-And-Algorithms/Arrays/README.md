\# Arrays



This folder contains C# implementation of basic array operations.



\## Topics Covered



\- Traversal

\- Insertion

\- Deletion

\- Searching



\## Array



An array is a linear data structure that stores elements of the same type in contiguous memory locations.



\## Advantages



\- Easy access using index

\- Simple to use

\- Efficient traversal



\## Disadvantages



\- Fixed size

\- Insertion and deletion can be costly
