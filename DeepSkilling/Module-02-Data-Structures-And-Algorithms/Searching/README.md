\# Searching Algorithms



This folder contains C# implementations of searching algorithms.



\## Algorithms Covered



\- Linear Search

\- Binary Search



\## Linear Search



Linear Search checks each element one by one until the required element is found.



Time Complexity: O(n)



\## Binary Search



Binary Search works on a sorted array. It repeatedly divides the search range into half.



Time Complexity: O(log n)
